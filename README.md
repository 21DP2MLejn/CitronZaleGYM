# TrenezieruZale
CitronZale is a imitation of Lemon Gym with my own added style.

# Tools used

Front-end stack:
  -  Vue.js 3
  -  HTML
  -  CSS
  -  Javascript

Back-end stack:
  - PHP
  - Laravel 


# Run this program

For front-end:
  - npm install
  - npm run dev

For back-end:
  - composer install
  - php artisan serve
